## Getting Started

  ### Dependencies
  #### Run ```setup.sh```
  This project requires the following dependencies:
  - make
  - g++
  - ncurses

  ### Installation
 To compile both the server and the client
  ```bash
  make
  ```
  To compile the server
  ```bash
  make server
  ```
  To compile the client
  ```bash
  make client
  ```
  To clean | To remove the compiled binaries
  ```bash
  make clean
  ```
